# Missions

Each mission is a JS file that exports mission data. Add new missions here.

## File format

See `content.js` (mission1 section) for the current format. Each mission needs:

- **title** — Mission name
- **client** — Who's hiring you
- **briefing** — Description shown when the job is offered
- **Character prompt additions** — What each relevant NPC knows about this mission (added to their system prompt in `characters.js`)
