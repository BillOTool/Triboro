    Schedule across viewer-time

    Each viewer's "Day 1" begins when they first open the page. Events run on one cadence (when each new headline drops); reactions run on another (how fast comments spill out under each headline). Tighter intervals = Twitch-stream feel; longer = marathon-storytelling.

      New story

      A story is an ordered sequence of events that tells a narrative thread. Events themselves stay independent — an event can belong to multiple stories or none. When you "schedule" a story, its events are laid out in that order and their reactions cascade beneath each one.
