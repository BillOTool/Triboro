# Art Assets

Drop artwork here. The game will reference files from these folders.

## Folders

- **backgrounds/** — Desktop wallpaper, window backgrounds, scene art
- **characters/** — NPC avatars and portrait art (named by character key, e.g. `benny.png`, `dale_voss.png`)
- **icons/** — App icons for the desktop (Hack Hub, Almanapp, TB News, Spoof Kit)
- **ui/** — UI elements, borders, decorative pieces

## Specs

- **Character avatars**: Square, ideally 128x128 or 256x256. PNG with transparency preferred.
- **Desktop icons**: 64x64 PNG with transparency.
- **Backgrounds**: 1920x1080 minimum. JPG or PNG.
- Keep file sizes reasonable — this runs in a browser.
