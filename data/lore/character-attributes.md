# Character Attributes

The data a target's Almanapp profile can expose. These double as the puzzle
surface: the fields a player scrapes, cross-references, and uses to socially
engineer a target.

- **Name**
- **Occupation**
- **Employer**
- **Education**
- **Birthday**
- **Group Affiliations**
- **Winkie Handle** — the resident's Almanapp handle/username
- **Neighborhood** — which floor/wing they live in
- **Last Login Location**
- **Age**
- **Marital Status**

## Puzzle hooks

- **Emergency Contact** — a lever for social-engineering a target's inner circle.
- **Path through the internet** — tracking a target's movement/activity trail
  across the Almanapp as a puzzle mechanic.

> Note: the running app's character frontmatter currently uses a lighter set
> (`name`, `handle`, `floor`, `avatar`, `faction`, `tags`). This fuller schema
> is the design target for hackable profiles.
